
x-ui start
x-ui stop
x-ui restart
x-ui status
x-ui settings
x-ui enable
x-ui log
x-ui banlog
x-ui update
x-ui setport
{start|stop|restart|status|settings|enable|disable|log|banlog|update|legacy|install|uninstall|setport}
